데이터 수집·갱신 · 시세·재무·거시·유니버스

흐름: 헬스 진단 → 개선 제안(승인) → 실측 확인. 제안 카드에 얕은 A/B(추정 composite IC 전후)를 붙여 방향을 보고, 심판은 승인 후 실측입니다. 수동 가중·백테스트는 필요할 때만 펼칩니다.

두뇌 레이어 · 엔진 헬스

소스→팩터→게이트→엔진→실측→진단→제안. 아래 카드에서 승인하면 엔진에 반영됩니다.

개선 제안 · 승인 후에만 엔진 반영 · 얕은 A/B

잘 안 맞는 팩터는 비중↓, 잘 맞는 팩터는 비중↑, 매수 적중률에 따라 문턱± — 모두 살짝만. “무엇이 바뀌는지·왜”만 보고 승인/반려하세요. 자동 적용 없음.
