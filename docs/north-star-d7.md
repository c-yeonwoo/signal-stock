# 북극성 지표 — D7

> 합의(2026-07-21): 셀프서브 개인투자자 · 수익화는 강의 부대(초대제 가능).  
> 기후는 뱃지 노출만(북극성과 무관).

## 한 줄

**D7 = 가입일(KST) 기준 이후 1~7일 안에 시그널 탭을 다시 연 유저 비율.**

## 정의 (고정)

| 항목 | 값 |
|---|---|
| 이벤트 | 로그인 세션에서 `GET /api/signals` **200** (시그널 탭 데이터 로드) |
| 방문 일 | 위 이벤트가 발생한 **KST 날짜** 1일 |
| D7 분자 | 가입일 D0 제외, D1~D7 중 방문 일이 **1일 이상**인 유저 |
| D7 분모 | 가입 후 8일 이상 경과한 유저(코호트 완성분) |
| 비포함 | 페이퍼·인사이트만 연 경우, 401, 관리자 refresh |

측정 인프라가 없으면 당분간 **수동**: 주간 코호트 초대 명단 × 서버 액세스 로그(또는 DB 세션)로 집계.  
자동 집계는 후속(이벤트 테이블 1개면 충분).

## 보조 지표 (북극성 아님)

| 지표 | 용도 |
|---|---|
| 매수 대기·근접 상세 클릭 | 빈 매수일 Empty UX 실험 (2~4주) |
| 페이퍼 봇 ON·체결 수 | 실습 깊이 |
| 코호트 가입→D7 | 강의 부대 가드 |

## 쓰지 않는 것

- 페이퍼 승률을 시그널 신뢰 헤드라인과 나란히 두는 것  
- “적중률 공개”를 D7 대리 지표로 쓰는 것(실측 성숙 전 금지)
